# 📝 Todo App (React)

A simple Todo application built with React to practice state management, component structure, and UI interactions.

---

## 🚀 Features

* Add new todos
* Edit existing todos (inline editing)
* Mark todos as completed
* Delete todos
* Persistent storage using `localStorage`
* Toast notifications for user feedback

---

## 🛠 Tech Stack

* React (Hooks)
* JavaScript (ES6+)
* CSS (custom styling, no frameworks)

---

## 📦 Installation

```bash
npm install
npm run dev
```
